# Customer Support Chatbot (Rasa)

## 📌 Overview
This project is a Customer Support Chatbot built using Rasa.  
It handles customer complaints across multiple domains such as billing, technical issues, delivery, and account management.

## 🚀 Features
- Intent classification for customer queries
- Handles multiple complaint categories:
  - Billing Issues
  - Refund Queries
  - Technical Issues
  - Delivery Issues
  - Account Issues
- Rule-based response system
- Structured dataset for complaint analysis
